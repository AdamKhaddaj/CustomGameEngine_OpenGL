Description

A short exploration game created in OpenGL that handles the entire rendering pipeline.

Custom vertex and fragment shaders in GLSL implement three term lighting model, screenspace effects, and particle systems that effectively use parallelization for efficient performance.

The Maze is procedurally generated each game, using a depth-first search algorithm on a geometry shader.

Created in collaboration with Leo McCarthy-Kennedy
